= GDG OIST Crafted Website
:navtitle: Welcome

Welcome to the internal engineering documentation for the GDG OIST project!

== Overview
This project powers the online presence for Google Developer Group (GDG) at OIST Bhopal. It is a fully dynamic, single-page application built with React, featuring a modern design system inspired by Google's brand colors, fluid animations, and a complete admin content management system backed by Supabase.

== Key Features 🚀
* *Dynamic Content*: Events, Galleries, Team Members, and Testimonials are fetched live from the database.
* *Admin Dashboard*: A protected admin panel for managing all dynamic content (events, gallery, team, testimonials).
* *Invite-Only Admin Access*: New admins can only join via invite tokens generated by existing admins.
* *Contact Form*: Validated submissions saved directly to the database with rate limiting and XSS protection.
* *Beautiful UI*: Built with `framer-motion` for fluid animations, `tailwindcss` for styling, and custom effects like Aurora WebGL backgrounds, floating blobs, and organic section dividers.
* *Dark/Light Mode*: Full theme support with `next-themes`.
* *Loading Screen*: Google-branded dot animation on first visit (session-cached).
* *Responsive Design*: Fully responsive with mobile-specific navigation and layouts.

== Site Sections
The landing page (`Index.tsx`) renders the following sections in order:

. *Hero* — Video background with parallax, animated heading, CTA buttons
. *TechMarquee* — Scrolling technology logos
. *About* — Community stats (icon + label cards), four pillars grid
. *Events* — Upcoming events fetched from database
. *Gallery* — Masonry photo gallery from database
. *Achievements* — Milestones via FlowingMenu + CircularGallery
. *Testimonials* — Rotating testimonial cards from database
. *Team* — Team member ProfileCards from database
. *FAQ* — Expandable accordion with common questions
. *Contact* — Validated contact form
. *CTABanner* — Call-to-action with gradient background
. *Footer* — Navigation links and social media

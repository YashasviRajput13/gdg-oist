== Overview
This project is built to power the online presence for Google Developer Group (GDG) at OIST. It handles displaying upcoming events, a dynamic photo gallery, team details, and collecting contact submissions. 

Unlike a static HTML website, this project is fully powered by a modern React frontend and hooked up to a Headless Content Management System (CMS) through Supabase.
